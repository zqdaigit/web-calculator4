# 简易 Web 计算器静态交互原型

直接打开 `index.html`，或通过任意静态服务器访问该目录。

## 覆盖范围

- 手机、平板、PC 三档响应式布局
- 明暗主题切换与本地偏好记忆
- 基础四则运算、百分比、正负切换、退格、C 与 AC
- 输入长度限制、浮点精度修正、科学计数法、除零与安全整数溢出锁定
- 物理键盘快捷键及按压期间的 active 反馈
- 会话态历史记录、结果回填、清空二次确认
- 点击结果复制及 2 秒 Toast 提示

## 静态模拟边界

- 不接入后端、账户体系或云端历史记录。
- 历史记录保存在浏览器 Session Storage；主题偏好保存在 Local Storage。
- 剪贴板能力受浏览器安全策略影响，原型包含兼容性回退。
